# BigInt

## BigInt

- JavaScript에서 큰 숫자를 사용할 때, `BigInt()`를 사용

```js
console.log(Number.MAX_SAFE_INTERGER); // 2^53 - 1

9007199254740991 + 1;
// 9007199254740992

9007199254740991 + 4;
// 9007199254740996

9007199254740991n + 1n;
// 9007199254740992n
// 제대로 나오게 하려면 n을 붙이세요.

BigInt(9007199254740991) + BigInt(2);
// 9007199254740993n
```
