# Sass

## Sass란?
- CSS 전처리기 중 하나
- Sass를 직접 작성하여 CSS로 변환하는 과정
- HTML태그의 계층처럼 Sass에서도 계층 구조로 작성
- 파일 확장자 `.scss`

:::info
### Sass의 장/단점

#### 장점:
- 계층적으로 표현(Nesting)으로 직관적이다.
- 변수, 믹스인(mixin) 등 편리하게 코드 작성가능하다.
- 파일을 분할하여 관리가 쉽다.

#### 단점:
- 계층의 깊이가 깊어질수록 코드 보기가 어려워진다.
- CSS개념 뿐만이 아닌 새로운 문법을 학습
:::
